## 关于 Cards Theme

Cards 是一个简洁优雅的 Hugo 主题，采用卡片式布局设计，专注于阅读体验。

### 特性

- **卡片式布局** - 清晰的内容组织方式
- **多语言支持** - 内置多种语言翻译
- **深色模式** - 自动/手动切换深色主题
- **自定义颜色** - 可自由配置主题色彩
- **搜索功能** - 基于 Fuse.js 的全文搜索
- **响应式设计** - 完美适配各种设备
- **SEO 优化** - 内置搜索引擎优化
- **图片懒加载** - 自动处理内容图片
- **数学公式** - 支持 KaTeX 数学渲染
- **Mermaid 图表** - 支持流程图、时序图等

### 技术栈

- Hugo 静态网站生成器
- 原生 JavaScript (无框架依赖)
- CSS 变量主题系统
